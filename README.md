# SCANNER-LIDAR-KABUM-SMART-700-DECIFRADO
Driver otimizado para o sensor LiDAR do robô KaBuM! Smart 700. Traduz o protocolo serial de 36-bytes em coordenadas cartesianas (X, Y) via Raspberry Pi Zero 2W. Solução acessível (sub R$100) para mapeamento SLAM e robótica DIY. Inclui transmissão via UDP para visualização em tempo real. 🚀
Este codigo ajuda pessoas que nao tem condições financeiras para adquirir um sensor lidar como o RPLidar A1, custando na casa dos R$900,00.
